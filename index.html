Setup Awal SIUJANG

Hubungkan ke database Supabase (gratis) untuk menyimpan data 700+ santri.

📋 Cara mendapatkan Supabase URL & Key (gratis, 5 menit):
  1. Buka supabase.com → daftar akun gratis
  2. Klik New Project → isi nama (mis: pesantren) → pilih region terdekat → Create
  3. Tunggu ~1 menit sampai project siap
  4. Klik ⚙️ Project Settings → API
  5. Salin Project URL dan anon public key
  6. Tempel di bawah → Simpan

Data tersimpan aman di server Supabase milik Anda sendiri.
